# 🧱 Daangn Clone - Backend (Node.js + Express)

당근마켓 클론 프로젝트의 백엔드입니다.

---

## 🚀 프로젝트 초기 세팅

```bash
mkdir backend
cd backend
npm init -y

npm install express cors dotenv
npm install -D nodemon

npm install -D eslint prettier eslint-config-prettier
```

## 개발 도구
```

코드 검사

npx eslint .

코드 자동 정리

npx prettier --write .
```
